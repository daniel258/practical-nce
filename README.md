# Practical Considerations for Negative Control Exposures (NCEs)

This repository contains the working manuscript and appendix for a paper on the **power and operating characteristics of the canonical negative control exposure (NCE) test** (adding an NCE to the outcome model and testing its coefficient). The key message is that (i) power need not increase with confounding strength, and (ii) when an additional common cause links the exposure and the NCE, conditioning on the exposure can **attenuate the NCE coefficient and inflate variance**, potentially yielding false reassurance.
